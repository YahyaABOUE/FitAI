# FitAI – AI Fitness Coach

## Description
FitAI is an AI-powered fitness coach that generates personalized workout
plans using user profiles, embeddings, and a Neo4j knowledge graph.

## Technologies Used
- FastAPI (Backend)
- Neo4j (Graph Database)
- OpenAI API (Embeddings & reasoning)
- Python
- HTML / CSS / JavaScript (Frontend)

## Architecture
- Frontend → FastAPI REST API
- Backend → Semantic matching + Neo4j
- AI → Personalized plan generation
